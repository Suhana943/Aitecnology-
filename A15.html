Singular Value Decomposition (SVD) Basics

By: Aitecnology Team | Published: July 9, 2026

Singular Value Decomposition (SVD) is one of the most powerful techniques in linear algebra. It acts as a "data compressor" that breaks down a complex matrix into three simpler, fundamental components, revealing the underlying patterns in your data.

1. What is SVD?

SVD factorizes any matrix A (of any size) into three distinct matrices: A = UΣVᵀ.

  • U (Left Singular Vectors): Captures the relationships between rows.
  • Σ (Singular Values): A diagonal matrix representing the "strength" or importance of each feature.
  • Vᵀ (Right Singular Vectors): Captures the relationships between columns.

2. Why use it?

  • Dimensionality Reduction: Removes noise by keeping only the largest singular values in Σ.
  • Data Compression: Represents huge datasets with far less memory.
  • Recommendation Engines: Predicts user ratings by finding latent patterns in user-item matrices.

3. How does it work?

SVD finds the directions (vectors) in your data where the variance is maximized. By sorting these directions by their singular values, you can discard noise and focus on the core signal.

4. Code Example

import numpy as np

# Create a sample matrix
A = np.array([[1, 2], [3, 4], [5, 6]])

# Perform SVD
U, S, Vt = np.linalg.svd(A)

print("U matrix:\n", U)
print("Singular values (S):\n", S)
print("Vt matrix:\n", Vt)
Aitecnology Tip: When working with sparse data (like user-movie ratings), always use Truncated SVD. It is faster and prevents the "Curse of Dimensionality" by focusing strictly on the most informative features!

Data Cleaning with Pandas

Published: July 9, 2026 | Category: Coding

Data cleaning is often the most time-consuming yet critical step in any data science project. Using the Pandas library, you can systematically handle messy, incomplete, or inconsistent datasets, ensuring your analysis is based on high-quality information.

Common Data Cleaning Tasks

Before you can analyze your data, you typically need to address several common issues:

  • Handling Missing Values: You can detect null values using .isnull() and either remove them with .dropna() or fill them with meaningful values (like the mean or median) using .fillna().
  • Removing Duplicates: Identify and eliminate redundant rows that could skew your results using the .drop_duplicates() method.
  • Type Conversion: Ensure your data is in the correct format (e.g., converting a string column containing numbers into numeric types) using .astype().
  • Renaming and Reformatting: Clean up column headers for better readability and standardize string data (like removing whitespace) using the .str accessor.

Quick Example

import pandas as pd
import numpy as np

# Sample data with missing values
df = pd.DataFrame({'Score': [85, np.nan, 90, 85], 
                   'Student': ['Alice', 'Bob', 'Charlie', 'Alice']})

# Remove duplicates and fill missing values
df = df.drop_duplicates()
df['Score'] = df['Score'].fillna(df['Score'].mean())

print(df)

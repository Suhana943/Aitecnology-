Supervised Learning: Regression Models

Published: July 9, 2026 | Category: ML

Regression analysis is a fundamental pillar of supervised learning. It is used to predict continuous numerical values based on input features. Unlike classification, which predicts discrete labels (e.g., "spam" or "not spam"), regression models estimate the relationship between variables to forecast quantities like prices, temperatures, or sales volume.

The Goal of Regression

In regression, the model learns a mapping function $f$ from input variables $X$ to a continuous output variable $y$. The objective is to minimize the error between the model's predicted output $\hat{y}$ and the actual observed values $y$. Common metrics for this error include Mean Squared Error (MSE) or Mean Absolute Error (MAE).

Popular Regression Algorithms

  • Linear Regression: The simplest form, modeling the relationship by fitting a linear equation to the observed data.
  • Polynomial Regression: Extends linear regression by adding polynomial terms, allowing the model to fit non-linear, curved relationships.
  • Ridge and Lasso Regression: Regularized versions of linear regression that add a penalty term to the cost function to prevent overfitting by shrinking coefficients.
  • Decision Tree and Random Forest Regressors: Non-linear models that predict values by partitioning the data into smaller, homogeneous groups.

Quick Example

from sklearn.linear_model import LinearRegression
import numpy as np

# Sample data
X = np.array([[1], [2], [3], [4]])
y = np.array([2, 4, 5, 4])

# Model training
model = LinearRegression()
model.fit(X, y)

# Prediction
prediction = model.predict([[5]])
print(f"Prediction for input 5: {prediction[0]}")

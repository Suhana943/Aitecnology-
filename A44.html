Support Vector Machines: Linear Classifiers

Published: July 9, 2026 | Category: ML

Support Vector Machines (SVM) are powerful supervised learning models used for classification and regression. The primary goal of a linear SVM is to find the optimal hyperplane that separates data points of different classes with the maximum possible margin.

How Linear SVM Works

Unlike logistic regression, which focuses on predicting class probabilities, SVM focuses on the geometric boundary between classes:

  • The Hyperplane: A flat decision boundary that separates the data into two distinct groups.
  • Support Vectors: These are the data points closest to the hyperplane. They are the "critical" points that define the position and orientation of the hyperplane.
  • The Margin: The distance between the hyperplane and the nearest support vector. SVM aims to maximize this margin.

Why Use Linear SVM?

  • High-Dimensional Efficiency: It is extremely effective in high-dimensional spaces.
  • Robustness: Because it relies only on support vectors, it is less sensitive to outliers.

Quick Example

from sklearn.svm import SVC
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split

# Generate linearly separable data
X, y = make_classification(n_features=2, n_redundant=0, n_informative=2, random_state=42)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Initialize and train linear SVM
# kernel='linear' ensures a straight-line boundary
clf = SVC(kernel='linear')
clf.fit(X_train, y_train)

print(f"Accuracy: {clf.score(X_test, y_test):.2f}")

Supervised Learning: Classification Algorithms

Published: July 9, 2026 | Category: ML

Classification is a core supervised learning task where the goal is to predict discrete category labels for new input data. Unlike regression, which predicts quantities, classification assigns observations to predefined groups based on learned patterns from historical data (e.g., classifying an email as "Spam" or "Not Spam").

The Goal of Classification

The objective is to train a model to find a decision boundary—a threshold or geometric surface—that effectively separates different classes in the feature space. Performance is typically evaluated using metrics like Accuracy, Precision, Recall, and the F1-Score, rather than error distance.

Popular Classification Algorithms

  • Logistic Regression: Despite the name, this is used for classification. It outputs the probability of an instance belonging to a specific class using the sigmoid function.
  • K-Nearest Neighbors (KNN): A simple, instance-based algorithm that classifies a data point based on the majority class of its 'K' closest neighbors.
  • Support Vector Machines (SVM): Finds the optimal hyperplane that maximizes the margin (distance) between different classes.
  • Decision Trees & Random Forests: Uses a tree-like model of decisions. Random Forests improve robustness by building multiple trees and aggregating their votes.

Quick Example

from sklearn.linear_model import LogisticRegression
from sklearn.datasets import make_classification

# Generate sample data
X, y = make_classification(n_features=2, n_redundant=0, n_clusters_per_class=1)

# Model training
clf = LogisticRegression()
clf.fit(X, y)

# Prediction
prediction = clf.predict([X[0]])
print(f"Predicted class for the first sample: {prediction[0]}")

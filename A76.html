Implementation: Q-Learning Algorithm

Niche Q-learning ka Python implementation diya gaya hai jo numpy aur gymnasium ka use karta hai:

import numpy as np
import gymnasium as gym

# Environment load karein
env = gym.make("FrozenLake-v1", is_slippery=False)

# Q-Table initialize karein
q_table = np.zeros((env.observation_space.n, env.action_space.n))

# Bellman Update Rule
new_value = old_value + alpha * (reward + gamma * next_max - old_value)

Key Concepts Explained:

  • Alpha (Learning Rate): Kitni jaldi agent nayi information ko purani info ke sath update karta hai.
  • Gamma (Discount Factor): Future rewards ki importance.

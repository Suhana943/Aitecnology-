Efficient Model Selection with GridSearch

Published: July 9, 2026 | Category: ML

Model selection is the process of finding the best "hyperparameters"—the configuration settings that are not learned by the model itself. GridSearch is a systematic, automated technique used to navigate this configuration space efficiently.

How GridSearch Works

GridSearch performs an exhaustive search over a specified subset of the hyperparameter space:

  • Define a Grid: Create a dictionary defining the parameters and values to test.
  • Cross-Validation: For every combination, the model is trained using K-Fold cross-validation to ensure reliable performance estimates.
  • Evaluate and Select: It calculates the average score for each combination and selects the best configuration.

Quick Example

from sklearn.model_selection import GridSearchCV
from sklearn.ensemble import RandomForestClassifier

# Define the model and parameter grid
rf = RandomForestClassifier()
param_grid = {
    'n_estimators': [50, 100, 200],
    'max_depth': [None, 10, 20]
}

# Run GridSearch
grid_search = GridSearchCV(estimator=rf, param_grid=param_grid, cv=5)
grid_search.fit(X_train, y_train)

print("Best parameters:", grid_search.best_params_)

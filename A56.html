PyTorch: Getting Started with Tensors

Published: July 9, 2026 | Category: Frameworks

At the heart of PyTorch lies the Tensor. It is a multi-dimensional matrix that can run on GPUs, enabling the massive parallel computation required for deep learning.

Tensor Basics

  • Scalar: Single value.
  • Vector: 1D array.
  • Matrix: 2D array.
  • Tensor: n-dimensional array.

Why Tensors?

Tensors allow for GPU acceleration and automatic differentiation, which are essential for training modern neural networks efficiently.

Quick Example

import torch

# Create a tensor
x = torch.tensor([[1, 2], [3, 4]])

# Perform operations
y = x * 2
print("Tensor x:\n", x)
print("Result of x * 2:\n", y)

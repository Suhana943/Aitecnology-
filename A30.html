Merging and Joining Datasets in Pandas

Published: July 9, 2026 | Category: Coding

In real-world data analysis, information is rarely contained within a single file. You often need to combine datasets from various sources to get a complete picture. Pandas provides powerful tools for this, mirroring the logic used in SQL databases.

Core Methods for Combining Data

Pandas offers two primary ways to combine DataFrames:

  • pd.merge(): This is the most versatile method. It allows you to perform database-style joins (inner, left, right, outer) based on common columns (keys).
  • pd.concat(): Use this when you want to stack DataFrames either vertically (adding more rows) or horizontally (adding more columns) along a particular axis.

Understanding Join Types

When using merge(), the how parameter determines which rows are kept:

  • Inner Join: Returns only the rows where the join key exists in both DataFrames.
  • Left Join: Returns all rows from the left DataFrame and matched rows from the right.
  • Right Join: Returns all rows from the right DataFrame and matched rows from the left.
  • Outer Join: Returns all rows when there is a match in either the left or right DataFrame.

Quick Example

import pandas as pd

df1 = pd.DataFrame({'Key': ['A', 'B'], 'Value1': [1, 2]})
df2 = pd.DataFrame({'Key': ['A', 'C'], 'Value2': [3, 4]})

# Inner join on 'Key'
merged_df = pd.merge(df1, df2, on='Key', how='inner')

print(merged_df)

Mastering Control Flow and Loops

Published: July 10, 2026

What is a Loop?

Imagine you need to say "Hello" to 100 people. Instead of writing 100 lines of code, you use a "Loop." A loop tells your program to perform the same task repeatedly until a specific goal is reached.

Types of Loops

1. For Loop (The "Counted" Loop)

Use this when you know exactly how many times the action should repeat.

for i in range(1, 6):
    print("Number:", i)

2. While Loop (The "Conditional" Loop)

This keeps running as long as a specific condition is True.

battery = 0
while battery < 100:
    print(f"Charging... {battery}%")
    battery += 20

Why are Loops important in AI?

  • Saves Time: Processes massive datasets in seconds.
  • Model Training: Used to run "epochs" in machine learning.
  • Automation: Handles repetitive tasks like image resizing.
Aitecnology Tip: Always include an exit condition to avoid an "Infinite Loop."

Computer Vision: Pooling and Downsampling

Published: July 9, 2026 | Category: CV

Pooling layers reduce the spatial size of feature maps, making CNNs more computationally efficient and more robust to small variations in input data.

Common Pooling Types

  • Max Pooling: Extracts the highest intensity feature in the region (most common).
  • Average Pooling: Takes the average of all features in the region for a smoother downsample.

Quick Example

# Define a 2x2 Max Pooling layer
pool_layer = nn.MaxPool2d(kernel_size=2, stride=2)

# Pool a 4x4 input to a 2x2 output
output = pool_layer(input_map)

Neural Networks: Activation Functions Overview

Published: July 9, 2026 | Category: Deep Learning

Activation functions are the mathematical "gatekeepers" of neural networks. They decide whether a neuron should be "activated" based on the weighted sum of its inputs plus a bias. Without them, a neural network would simply be a linear regression model, regardless of how many layers it has.

Popular Activation Functions

  • Sigmoid: Maps values between 0 and 1. Rarely used in hidden layers today due to vanishing gradients.
  • Tanh: Maps values between -1 and 1. Zero-centered, which can speed up training.
  • ReLU: The standard for hidden layers. Returns the input for positive values and zero for negative ones.
  • Softmax: Used in output layers for multi-class classification to turn scores into probabilities.

Why Non-Linearity Matters

Without activation functions, deep networks are just complex linear transformations. Non-linearity allows the network to learn and represent complex patterns in data.

Quick Example

import numpy as np

def relu(x):
    return np.maximum(0, x)

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

# Example input
data = np.array([-2, -1, 0, 1, 2])
print("ReLU:", relu(data))
print("Sigmoid:", sigmoid(data))

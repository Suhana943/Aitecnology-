Gradient Descent Fundamentals

By: Aitecnology Team | Published: July 10, 2026

Gradient Descent is the "compass" of machine learning. It is an optimization algorithm used to train models by iteratively adjusting their internal weights to minimize error.

1. Core Concepts

The algorithm calculates the gradient (slope) of the loss function and takes small steps downhill using a learning rate to update parameters.

2. Challenges in Training

  • Vanishing Gradients: Gradients become too small in deep networks, stopping learning.
  • Exploding Gradients: Gradients become too large, causing unstable model updates.
  • Local Minima/Saddle Points: Getting stuck in non-optimal valleys or flat regions.

3. Modern Variations

  • Momentum: Adds "velocity" to the update to push through flat regions.
  • Adam Optimizer: Adapts learning rates for each parameter, currently the industry standard.

4. Python Implementation

import torch.optim as optim

# Standard SGD (Vanilla Gradient Descent)
optimizer_sgd = optim.SGD(model.parameters(), lr=0.01)

# Adam Optimizer (Includes Momentum and Adaptive Learning Rates)

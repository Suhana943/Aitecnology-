PyTorch: Automatic Differentiation with Autograd

Published: July 9, 2026 | Category: Frameworks

torch.autograd is PyTorch’s automatic differentiation engine. It constructs a dynamic computational graph to trace operations and calculate gradients automatically, enabling seamless backpropagation.

How It Works

  • Graph Tracking: When requires_grad=True, PyTorch records every operation.
  • Chain Rule: Upon calling .backward(), the engine traverses the graph backwards to calculate derivatives.
  • Dynamic Nature: The graph is built on the fly, allowing for flexible Python logic during training.

Quick Example

import torch

x = torch.tensor(2.0, requires_grad=True)
y = torch.tensor(3.0, requires_grad=True)
z = x**2 * y

z.backward()
print("Gradient of x:", x.grad) # 12.0
print("Gradient of y:", y.grad) # 4.0

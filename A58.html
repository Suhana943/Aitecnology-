PyTorch: Optimizer and Loss Functions

Published: July 9, 2026 | Category: Frameworks

In PyTorch, the Loss Function measures model error, while the Optimizer performs the weight updates based on that error.

Core Components

  • Loss Function: Quantifies how far off predictions are (e.g., CrossEntropyLoss for classification).
  • Optimizer: Applies mathematical rules to update weights (e.g., Adam, SGD).

Quick Example

criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

# Training loop step:
optimizer.zero_grad()
loss = criterion(model(inputs), targets)
loss.backward()
optimizer.step()

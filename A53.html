Neural Networks: Feedforward Architectures

Published: July 9, 2026 | Category: Deep Learning

Feedforward Neural Networks (FNNs) are the foundation of deep learning. Information moves in only one direction—forward—through the input, hidden, and output layers.

Architecture Components

  • Input Layer: Entry point for raw features.
  • Hidden Layers: Compute internal representations to extract patterns.
  • Output Layer: Produces the final prediction.
  • Weights/Biases: Learnable parameters that determine the network's function.

Quick Example

import torch.nn as nn

class FeedforwardNet(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(FeedforwardNet, self).__init__()
        self.hidden = nn.Linear(input_size, hidden_size)
        self.relu = nn.ReLU()
        self.output = nn.Linear(hidden_size, output_size)
        
    def forward(self, x):
        x = self.relu(self.hidden(x))
        return self.output(x)

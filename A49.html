Dimensionality Reduction: t-SNE for Visualization

Published: July 9, 2026 | Category: ML

t-Distributed Stochastic Neighbor Embedding (t-SNE) is a sophisticated dimensionality reduction technique specifically designed for visualizing high-dimensional datasets. Unlike PCA, which focuses on preserving global variance, t-SNE is adept at preserving local structure, making it exceptionally good at revealing clusters and patterns in complex data.

How t-SNE Works

t-SNE treats dimensionality reduction as a probability distribution matching problem:

  • High-Dim Similarity: Calculates probabilities of points being neighbors in the original space.
  • Low-Dim Similarity: Calculates corresponding probabilities in the target 2D/3D space.
  • KL Divergence: Uses gradient descent to minimize the difference between these two probability distributions.

When to Use t-SNE

  • Data Exploration: Perfect for visualizing complex, non-linear data structures.
  • Pattern Discovery: Helps identify subgroups and outliers that aren't obvious in high dimensions.

Quick Example

from sklearn.manifold import TSNE
from sklearn.datasets import load_digits

# Load high-dimensional data
digits = load_digits()
X = digits.data

# Reduce to 2 dimensions for visualization
tsne = TSNE(n_components=2, perplexity=30)
X_embedded = tsne.fit_transform(X)

print("Original shape:", X.shape)
print("Embedding shape:", X_embedded.shape)

Dimensionality Reduction: Principal Component Analysis

Published: July 9, 2026 | Category: ML

Principal Component Analysis (PCA) is a powerful technique for reducing the number of variables in a dataset while preserving as much of the original variance (information) as possible. It is widely used to combat the "curse of dimensionality," improve visualization, and speed up machine learning models.

How PCA Works

PCA transforms the original features into a new set of uncorrelated variables called Principal Components. The process involves:

  • Standardization: Scaling features to have mean 0 and variance 1.
  • Covariance Calculation: Determining how variables relate to each other.
  • Eigen-decomposition: Identifying the directions (eigenvectors) that capture the most variance.
  • Projection: Mapping original data onto the most important components.

When to Use PCA

  • Data Visualization: Plotting high-dimensional data in 2D or 3D.
  • Noise Reduction: Filtering out low-variance components.
  • Addressing Multicollinearity: Removing dependencies between input features.

Quick Example

from sklearn.decomposition import PCA
from sklearn.datasets import load_iris

# Load sample data
data = load_iris()
X = data.data

# Reduce to 2 dimensions
pca = PCA(n_components=2)
X_reduced = pca.fit_transform(X)

print("Original shape:", X.shape)
print("Reduced shape:", X_reduced.shape)
print("Explained variance ratio:", pca.explained_variance_ratio_)

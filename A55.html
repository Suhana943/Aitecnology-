Stochastic Gradient Descent Optimization

Published: July 9, 2026 | Category: Deep Learning

Stochastic Gradient Descent (SGD) is the iterative algorithm used to update neural network weights. By using small, random subsets of data, it accelerates training and helps models avoid getting stuck in suboptimal solutions.

Why "Stochastic"?

Instead of calculating the gradient for the entire dataset (which is slow), SGD uses a single data point or a mini-batch. This randomness allows the algorithm to explore the loss landscape more efficiently and overcome local minima.

Key Concepts

  • Learning Rate: Controls the size of the steps taken toward the minimum.
  • Mini-batching: The practical middle ground between pure SGD and batch descent.
  • Momentum: An enhancement that adds velocity to weight updates to speed up convergence.

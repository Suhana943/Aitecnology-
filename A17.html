Applying the Chain Rule in Calculus

By: Aitecnology Team | Published: July 9, 2026

The Chain Rule is a fundamental formula used to find the derivative of a "composite function." If a function is nested inside another, the Chain Rule allows you to calculate the rate of change for the entire structure by working from the outside in.

1. What is the Chain Rule?

If you have a function y = f(g(x)), the Chain Rule states:

dy/dx = df/dg * dg/dx

In simple terms: The derivative of the whole is the derivative of the outer function multiplied by the derivative of the inner function.

2. Why is it Essential?

  • Complex Modeling: Helps solve real-world systems where processes are chained together.
  • Deep Learning: The foundation of Backpropagation. It allows us to calculate how a change in the output propagates back through layers to update weights.

3. How to Apply It

  1. Identify the outer function (f) and inner function (g).
  2. Differentiate the outer function, keeping the inner part unchanged.
  3. Multiply by the derivative of the inner function.

4. Code Example

import sympy as sp

# Define variable
x = sp.symbols('x')

# Define composite function: (3x^2 + 1)^5
y = (3*x**2 + 1)**5

# Calculate derivative
dy_dx = sp.diff(y, x)

print(f"The derivative is: {dy_dx}")
# Result: 5*(3*x^2 + 1)**4 * (6*x)
Aitecnology Tip: When solving manually, always write down u = g(x) explicitly. It prevents "nested function fatigue" and helps you track which part of the function you have already differentiated!
